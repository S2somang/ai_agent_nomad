구직자의 이력서를 바탕으로 최적의 채용 공고를 찾고, 이력서를 최적화하며, 면접 준비까지 도와주는 자동화 프로세스

1. 채용 정보 추출 태스크 (job_extraction_task)
역할: 웹 서핑을 통해 조건에 맞는 실제 채용 공고 데이터를 수집합니다.

주요 작업: 특정 위치, 직급, 직무에 맞는 공고를 검색하고 리스트화합니다.

번역: "특정 위치의 {level} 레벨 {position} 채용 공고를 검색하고 추출하십시오."

2. 채용 매칭 태스크 (job_matching_task)
역할: 수집된 공고들과 사용자의 이력서를 비교하여 점수를 매깁니다.

주요 작업: 기술 스택, 경력 수준, 선호도(원격 근무 등), 연봉 등을 분석하여 1~5점 사이의 match_score를 부여하고 그 이유를 작성합니다.

번역: "추출된 채용 리스트와 사용자의 이력서를 비교하여 각 공고의 적합성 점수와 이유를 기술하십시오."

3. 최종 직무 선택 태스크 (job_selection_task)
역할: 매칭된 공고 중 사용자에게 가장 베스트인 '단 하나'의 공고를 결정합니다.

주요 작업: 점수가 가장 높은 공고를 선택하고, 왜 이 직무가 사용자에게 최적인지 최종 정당성을 부여합니다.

번역: "점수와 이유를 분석하여 사용자의 목표와 가장 잘 맞는 단 하나의 공고를 최종 선택하십시오."

4. 이력서 재작성 태스크 (resume_rewriting_task)
역할: 선택된 특정 직무에 합격할 확률을 높이도록 이력서를 맞춤형으로 수정합니다.

주요 작업: 사실을 왜곡하지 않는 범위 내에서, 해당 직무의 키워드와 요구사항에 맞춰 강조할 프로젝트와 기술을 재배치합니다.

번역: "선택된 직무의 요구사항에 맞춰 사용자의 실제 경력을 바탕으로 이력서를 최적화하여 재작성하십시오."

5. 기업 조사 태스크 (company_research_task)
역할: 면접을 대비해 해당 기업에 대한 깊이 있는 정보를 수집합니다.

주요 작업: 기업의 미션, 가치, 최근 뉴스, 팀 구조 등을 조사하고 면접 예상 질문과 역질문 리스트를 만듭니다.

번역: "선택된 채용 기업의 산업, 미션, 가치, 최근 뉴스 등을 조사하고 면접 전략을 세우십시오."

6. 면접 준비 태스크 (interview_prep_task)
역할: 앞선 모든 정보를 종합하여 최종 '면접 대비 전략 가이드'를 제작합니다.

주요 작업: 직무 개요, 이력서 하이라이트, 기업 요약, 예상 질문 및 답변 전략을 포함한 종합 문서를 작성합니다.

번역: "최종 선택된 직무, 최적화된 이력서, 기업 조사 내용을 결합하여 상세한 면접 준비 문서를 작성하십시오."
